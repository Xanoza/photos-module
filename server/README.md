## Available API Routes

| Endpoint            | Type    | Operation                 |
|---------------------|---------|---------------------------|
| `/api/photos/:id`   | GET     | Get a photo               |
| `/api/photos/`      | POST    | Add a photo               |
| `/api/photos/:id`   | PUT     | Update photo              |
| `/api/photos/:id`   | DELETE  | Delete photo              |
